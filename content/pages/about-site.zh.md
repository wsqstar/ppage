# 关于本站

本站点使用 **PPage** 构建，这是一个纯前端的个人主页生成系统。

## 主要特性

- 🚀 **纯前端架构** - 基于 React + Vite，无需后端服务器
- 📝 **Markdown 驱动** - 支持 Markdown 内容创作，自动解析和渲染
- 🎨 **多主题支持** - 内置 Light、Dark、Academic 三种主题
- 🌍 **国际化** - 支持中英文切换，内容可配置
- ⚙️ **配置驱动** - 通过 YAML 配置文件快速定制
- 📦 **自动发现** - 自动扫描 Markdown 文件和资源链接
- 🎯 **GitHub Pages 部署** - 一键部署到 GitHub Pages

## 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **Markdown 渲染**: markdown-it
- **路由**: React Router
- **样式**: CSS Modules
- **部署**: GitHub Pages

## 适用场景

- 个人主页/简历展示
- 学术研究者个人网站
- 博客/文章发布
- 项目作品集展示
- 文件/论文分享
